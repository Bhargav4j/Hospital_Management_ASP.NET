# Hospital Management System - .NET 8

A modern hospital management system built with .NET 8 and clean architecture, migrated from ASP.NET Web Forms.

## Technology Stack
- .NET 8
- ASP.NET Core Razor Pages
- Entity Framework Core 8.0
- SQL Server
- AutoMapper
- Serilog

## Build
```bash
dotnet build HospitalManagement.sln
```

## Run
```bash
dotnet run --project src/HospitalManagement.Web
```

## Migration Status
Successfully migrated from ASP.NET Web Forms 4.5.2 to .NET 8 with clean architecture.
See docs/MIGRATION_NOTES.md for details.
